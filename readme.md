# CEGEPCI_Traitement_donnees

Ce dossier regroupe les **données brutes issues du drone**, les **rapports de mesures**, ainsi que les **scripts de traitement et de création de jeux de données**.  
Le dossier est destiné à être hébergé sur un dépôt Git séparé.

## 📁 Arborescence du dossier

### rapport
Contient les différents **rapports de mesures**.

#### rapports_csv
Anciens rapports réalisés lors de l’**initialisation du stage**.  
Ces fichiers ne sont plus utilisés et ont peu d’importance pour le projet actuel.

#### rapports_xlsx
Rapports au format Excel :
- **Q40** : rapports utilisés pour la **création du jeu de données**.  
  Ils correspondent aux **prises de mesures effectuées au port**.
- **Vieux port** : anciens rapports issu de l’initialisation du projet, **peu pertinent** pour les analyses actuelles.

#### moyennes_paliers.csv
Première version (**v1**) d’un fichier représentant ce à quoi pourrait ressembler le **dataset final**, basé sur des moyennes par paliers.

### results
Résultats des **tests et analyses effectués sur les données**.

### scripts
Scripts utilisés pour :
- Les tests
- Le traitement des données
- La création des datasets
